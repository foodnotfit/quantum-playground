## 🚀 How to Download & Run (Step by Step)

### Option A — No Git Required (Easiest)

1. Go to: **https://github.com/foodnotfit/quantum-playground**
2. Click the green **`<> Code`** button
3. Click **`Download ZIP`**
4. Unzip the downloaded file anywhere on your computer
5. Open the unzipped folder
6. Double-click **`index.html`**
7. It opens in your browser — **done!**

> ✅ No internet required after download. Works 100% offline.
> ✅ No Python, Node, or any software needed.
> ✅ Works on Windows, Mac, and Linux.

---

## 🔧 Troubleshooting

**App won't open / blank page?**
- Make sure you're opening `index.html` directly in a browser, not a text editor
- Try right-clicking `index.html` → Open With → Chrome/Firefox/Safari

**Buttons not responding?**
- Make sure JavaScript is enabled in your browser (it is by default)
- Try refreshing the page (Ctrl+R / Cmd+R)

**Looks wrong on mobile?**
- Rotate to landscape for best experience on phones
- Works best on tablets and computers
