# Quantum Playground

Interactive quantum computing education app for DARPA Family Day 2026.

## How to run
Open `index.html` in any browser. No install needed. Works offline.

## Games
1. 🪙 Coin Flip — Superposition
2. ⚛️ Entanglement Lab — Quantum entanglement
3. 🏁 Quantum Race — Quantum parallelism
4. 🔐 Code Breaker — Quantum vs classical cryptography
